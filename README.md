# Tucil 1 - Stima
A solver with brute force algorithm to solve Queens LinkedIn Game

## Usage

Run the application:
```
python src/main.py
```

Or use the compiled executable in the `bin` folder.

## Building

To compile the application:
```
python -m PyInstaller --onefile --windowed --distpath bin src/main.py
```
